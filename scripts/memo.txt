Sample CoM combinations:
python sample_com_combinations.py
Visualize sampled CoM combinations:
python visualize_sampled_com_combinations.py

Check if the queried model is correct
check_model.py

Procedure to generate ground truth:
1. sample contact transitions of a specific environment type:
python contact_transition_sampler.py -a: [environment_type] -b: [environments_file] -c: [transitions_file]
2. check transition type of the transitions in a specific file:
python check_transitions_type.py
3. compute dynamic cost for the transitions in the environments of a specific environment type
python compute_ddyn.py -t: [environment_type]
4. find the minimal dynamic cost and generate depth maps for environments
python generate_depth_map.py
